# gcl_iam

`gcl_iam` добавляет аутентификацию и авторизацию в Python-сервисы на
RESTAlchemy. Библиотека проверяет JWT, получает данные пользователя и разрешения
через IAM-драйвер, сохраняет их в контексте запроса и применяет в контроллерах и
при сериализации полей.

## Возможности

- Проверка JWT с HS256 и RS256.
- Поддержка предыдущего ключа при ротации.
- Получение JWKS и introspection с удалённого IAM endpoint.
- Пользовательские IAM-драйверы внутри процесса.
- Разрешения формата `service.resource.action` с `*` на любом уровне.
- Ограничение проектных ресурсов по `project_id` токена.
- Защита глобальных и вложенных ресурсов.
- Обязательная проверка OTP для выбранных операций.
- Управление видимостью полей RESTAlchemy через IAM-правила.
- Доступ к токену, пользователю, проекту и разрешениям через текущий контекст.
- Единообразные HTTP-ответы для IAM-исключений.

`gcl_iam` не выдаёт учётные данные, не определяет роли приложения и не содержит
IAM-базу. За это отвечает IAM-сервис или пользовательский драйвер.

## Поддерживаемое окружение

- Python 3.10 или новее
- RESTAlchemy 15.x
- PyJWT 2.x
- JWT-подписи HS256 или RS256

Актуальные диапазоны зависимостей указаны в `pyproject.toml`.
